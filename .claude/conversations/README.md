# Claude Conversation Logs

This directory contains detailed logs of conversations with Claude, organized in a todo listicle format for easy tracking and reference.

## File Naming Convention
- Format: `YYYY-MM-DD-session-NNN.md`
- Example: `2025-01-21-session-001.md`

## Log Structure
Each conversation log includes:
1. **Summary** - Brief overview of conversation goals
2. **Tasks Completed** - Checkbox list of all tasks with status
3. **Commands Executed** - Key commands run during session
4. **Files Modified** - List of changed files with descriptions
5. **Key Decisions** - Important choices and reasoning
6. **Follow-up Items** - Future tasks to consider
7. **Notes** - Additional context

## Usage
- Review past conversations to understand project history
- Track completed tasks and pending items
- Reference commands and configurations
- Maintain continuity across Claude sessions

## Automation
Consider using git hooks to automatically generate conversation summaries based on commits and Claude interactions.
